This game is our project in school mocking the famous Space Invaders game. It is built on the XNA  framework in visual studio. 
The first player can move with the arrow keys and with the mouse and can shoot with thee up arrow or the left mouse button.
The second player can move with the G  and D keys and shoot with the R key. 
Menus choice change with up and down arrows and value changee with pgDown and pgUp keys.
